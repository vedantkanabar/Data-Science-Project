# Choudhray Dataset

- This is our real life data set from the Western part of USA.
- The file `choudhury_dataset_cleanup.py` creates a cleaned `choudhury_dataset_cleaned.csv` from `choudhury_dataset.csv`. This cleans up some of the formats in the data.
- It also creates two files `categories.txt` and `jobs.txt` containing the uniqe categories and jobs respectively.
