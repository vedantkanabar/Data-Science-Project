# Model Training and Evaluation (and tuning)

- This folder contains code for creating and tuning our model
- The `model_tuning.py` file contains code to tune hyperparameters for our models and stores the result in `tuning.txt`.
- The `model_training_evaluation.py` file contains code to read the features from `../combined_dataset/combined_encoded.csv` and train the model based on the features.
- The numeric metrics are in `results.txt` and the confusion plots are in files `Confusion Matrix - <Model>.png` and the feature importance plots are in `Feature_Importance_<Model>.png`.
